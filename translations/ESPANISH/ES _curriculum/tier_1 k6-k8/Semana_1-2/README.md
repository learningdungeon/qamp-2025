
# Semana 1 — Qubits, Superposición y Medición (Nivel 1, Grados 6–8)

**Idea principal:** Un qubit puede comportarse como una **moneda** que está en reposo (siempre igual) o como una **moneda en medio del giro** (indefinida), y leerlo es como **atrapar y mirar**.

## Lo que harás
- **Siempre igual:** lee una moneda en reposo (qubit sin puertas) → el gráfico de barras muestra 100% “0”.
- **Moneda cuántica:** agrega **H** (piensa: “mezclar”) → el gráfico de barras muestra aproximadamente **50% 0 / 50% 1**.
- **Inclina las probabilidades:** agrega una pequeña **puerta Tilt** (usa **RY** con un ángulo pequeño) → logra “mayoría 0” o “mayoría 1”.
- **Predice → Prueba → Explica → Itera:** para cada cambio, escribe tu predicción, ejecuta y luego explica el gráfico de barras.

> Aprendemos el **concepto**, no las matemáticas complicadas. Usaremos probabilidades simples como 50/50 o 70/30.
