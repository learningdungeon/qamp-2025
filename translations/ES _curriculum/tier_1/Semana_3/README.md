
# Semana 3 — Entrelazamiento y Estados de Bell (Nivel 1, Grados 6–8)

**Idea principal:** dos “dados mágicos” (qubits) pueden estar conectados de manera que sus resultados coincidan (o siempre sean opuestos).

## Lo que harás
- Construye un equipo Bell: **H en el primer qubit → CNOT al segundo → mide ambos**.
- Observa **resultados coincidentes** ~50% “00” y ~50% “11”.
- Haz un pequeño cambio (**X en el segundo qubit**) y observa **resultados opuestos** ~50% “01” y ~50% “10”.
- Usa una cadena corta de bits compartida para **revelar una pequeña imagen de 4×4 píxeles** con una regla simple de “igual/diferente”.

> Estamos aprendiendo el concepto, no las matemáticas complicadas. Piensa en monedas y dados, no en ecuaciones.
