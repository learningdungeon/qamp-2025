
# Guía para el Docente — Semana 3 (Nivel 1)

**Tono:** cálido, motivador, concreto. Usa comparaciones cotidianas (monedas, dados mágicos).  
**Objetivo:** que los estudiantes puedan construir un par de Bell, invertir el patrón con una puerta y explicar sus elecciones.

## Flujo (≈35–40 min)
1) Gancho (2): “Dos monedas que siempre coinciden—¿truco o tecnología?”  
2) Demostración (5): H(q0) → CNOT(q0→q1) → medir → mostrar barras 00/11.  
3) Explora (10): estudiantes replican.  
4) Remezcla (8): agrega X en q1 → muestra 01/10; compara con “sin CNOT”.  
5) “Postales” de píxeles (8): parejas usan una cadena compartida de 16 bits + un cifrado dado para revelar un ícono de 4×4.  
6) Compartir (2): parejas explican “qué cambié y por qué” (predice → prueba → explica → itera).

**Configuraciones**
- Histogramas Bell: Shots=1024, mide ambos qubits.
- Para una “clave” de 16 bits: Shots=1, ejecuta 16 veces; cada compañero conserva su bit.
- Lenguaje a usar: “resultados vinculados”, “coincidentes/opuestos”, “regla simple (igual=0, diferente=1)”.
